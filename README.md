# 🎬📈 Movie Recommender & Stock Price Prediction System

## 🚀 Overview
This project combines two machine learning applications:
1. Movie Recommendation System for personalized suggestions  
2. Stock Price Prediction model using historical data  

Developed as part of AICT Internship Task.

- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
